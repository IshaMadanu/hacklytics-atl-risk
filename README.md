# ATLRisk
The purpose of ATLRisk is to help people in Atlanta take control of their personal safety by giving them the information they need to make smarter, more informed decisions.

Running ATLRisk:
1. Clone the Repo:

```
git clone https://github.com/alisonkelton/hacklytics2026.git
cd hacklytics2026
```
2. Install dependencies

```
pip install -r requirements.txt
```

3. Get a Gemini API Key

Go to https://aistudio.google.com/apikey

Create a free API key

4. Create a .env file in the project root

```
GEMINI_API_KEY=your_api_key_here
```

5. Run the app
```
python app.py
```
